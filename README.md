# 🎨 MotionCSS AI

Gerador de animações CSS a partir de linguagem natural utilizando Inteligência Artificial.

## 🚀 Sobre o projeto

O MotionCSS AI permite transformar descrições simples como:

> "bola azul quicando"

em animações CSS prontas para uso, automatizando o processo de criação de animações e aumentando a produtividade de desenvolvedores front-end.

## 🧠 Como funciona

O usuário descreve a animação desejada em linguagem natural, e a aplicação utiliza APIs de Inteligência Artificial para gerar automaticamente o código CSS correspondente.

## 🛠️ Tecnologias utilizadas

- JavaScript
- Node.js
- APIs de Inteligência Artificial
- HTML & CSS
